Lina

„Hey Lina" – und das Leben wird ein bisschen leichter.

Lina ist eine Open-Source-Sprachassistentin für Android, gebaut für blinde und sehbehinderte Menschen. Kein Bildschirm nötig, kein Login, keine Cloud-Pflicht: Die Kernfunktionen laufen komplett auf dem Gerät. Getragen wird das Projekt von einem gemeinnützigen Verein.

Quellcode auf GitHub

Was Lina kann

  • Anrufe: Kontakte anrufen, eingehende Anrufe annehmen oder ablehnen – nur mit der Stimme.
  • Nachrichten schreiben und vorlesen: SMS senden, empfangene Nachrichten vorlesen lassen, direkt antworten.
  • Zeitung hören: Nachrichten aus vertrauten Quellen – erst die Zusammenfassung, auf Wunsch der ganze Artikel.
  • Hörbücher: Eigene Bibliothek und kostenlose Librivox-Hörbücher, mit Schlaf-Timer.
  • Einfach reden: Optional beantwortet Lina freie Fragen und unterhält sich – wie eine gute Bekannte.

So klingt das

„Hey Lina, ruf Boris an." – „Ich rufe Boris an."

„Hey Lina, was gibt es Neues?" – „Drei Meldungen. Erstens: …"

„Hey Lina, spiel mein Hörbuch weiter." – „Weiter geht's bei Kapitel vier."

Warum noch eine Assistentin?

Siri, Alexa und Co. sind für sehende Menschen gebaut – Bildschirm-Dialoge, visuelle Rückfragen, begrenzte Antworten. Lina ist von Grund auf Voice-First: Jede Aktion wird gesprochen bestätigt, nichts setzt Sehen voraus. Und weil sie Open Source ist, kann jede Community sie an ihre Bedürfnisse anpassen.

Technik in einem Satz

Weckwort-Erkennung (OpenWakeWord), Spracherkennung (Whisper) und Sprachausgabe (Piper) laufen offline auf dem Tablet; für freie Gespräche kann optional die Claude-API angebunden werden. Alles unter Apache-2.0-Lizenz.

Mitmachen

Lina sucht Mitstreiter:innen – Entwickler:innen, Tester:innen und besonders Menschen mit eigener Seheinschränkungs-Erfahrung. Der gesamte Code, die Dokumentation und die offenen Aufgaben liegen auf GitHub.
